Oversold · 超跌板块

底部反转选股 · 四大经典信号

扫描 全部 A 股（沪 / 深 / 创 / 科，仅排除 ST），用四个互补的底部反转技术信号分栏命中。所有信号支持 近 N 天内出现 的可调回溯窗口（下方"信号回溯天数"控制，默认 5 天）：
栏目1 「MACD 低位金叉」（最强信号 · 精选少量）：近 N 天内 MACD 红柱由负转正 + DIF 在零轴下方金叉（确保是底部金叉而非高位金叉）+ 当前 hist 仍 > 0 + RSI 25-65 健康反弹区 + 收盘价 ≤ MA60×1.02 (深度调整后) + 当日量 ≥ 5 日均量 + 当日收阳。
栏目2 「均线初翻多」（中信号 · 趋势启动）：近 N 天内 MA5 上穿 MA10 + 收盘价站上 MA10 + 距离 MA60 在 -20% ~ +5% + 近 20 日 RSI 曾 < 35 + 当前 RSI 35-65 + 当日收阳。
栏目3 「布林下轨反弹」（宽口径 · 早期信号）：近 N 日内有触及或跌破布林下轨 + 当日回到带内 + 收盘 < 中轨 MA20 + 当日 RSI > 5 日前 RSI + 当日收阳。
栏目4 「RSI 选股」（最宽 · 单一指标）：近 N 日内 RSI(14) 至少出现 1 次 < 30。最纯粹的"超卖"信号，不要求反弹/均线/量能配合。
每股票一旦选入即持久化保存到本地，记录选入时间 / 选入价；之后即便不再满足条件也不会丢失。每行显示现价 / 收益 / 最大涨跌 / 3 日峰值 / 操作建议，可单独 × 移除或一键 🗑 清空整列。命中股票自动传送到看板「底部反转」栏。涉及全 A 股扫描，首次加载约 5-10 分钟，结果缓存 60 分钟。
